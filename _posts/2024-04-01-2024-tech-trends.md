- 로컬 LLM의 성장

```python
# LLM 활용 예시
import openai

def generate_code(description):
    response = openai.ChatCompletion.create(
        model="gpt-4",
        messages=[{"role": "user", "content": description}]
    )
    return response.choices[0].message.content
```
