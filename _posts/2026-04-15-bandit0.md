## Bandit 0

![bandit0](/assets/images/bandit/bandit0.png)

### 접속

```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220
```

### 명령어

`ssh`: 원격 서버에 Secure Shell로 접속

`-p 2220`: SSH 포트 지정

### 비밀번호

`bandit0`

### 풀이

```bash
ls
cat readme
```

### 명령어

`ls`: 현재 디렉토리 파일/폴더 목록 출력

`cat <file>`: 파일 내용을 그대로 출력(읽기)

### 1033c "저 c 는 뭔데요"

`c`는 **bytes**로 지정한다는 의미

`b`: 512바이트 블록 (기본값, 생략 시 적용)

`c`: 1바이트 (Character/Byte)

`w`: 2바이트 (Word)

`k`: 킬로바이트 (1,024바이트)

`M`: 메가바이트 (1,048,576바이트)

`G`: 기가바이트 (1,073,741,824바이트)

정확히 `1033bytes` 인 파일을 찾으라는 명령입니다.

## Bandit6

![alt text](/assets/images/bandit/image-15.png)

이번 문제는 전체 파일 시스템에서 특정 조건의 파일 찾기이다.

- 소유자: `bandit7`
- 그룹: `bandit6`
- 크기: `33 bytes`

```bash
find -user bandit7 -group bandit6 -size 33c
```

찾아보니까 `denied` 저것들 오류 메시지인데 `2>/dev/null` 이걸 뒤에 추가하면 오류 메시지를 없앨 수 있다.

![alt text](/assets/images/bandit/image-16.png)

이렇게 나오는데 중간에 수상한 것이 보인다.

```bash
cat ./var/lib/dpkg/info/bandit7.password
```

<script>
  (() => {
    const pager = document.querySelector('[data-bandit-pager]');

    if (!pager) {
      return;
    }

    const pages = Array.from(pager.querySelectorAll('[data-bandit-page]'));
    const tabs = Array.from(pager.querySelectorAll('[data-bandit-tab]'));
    const status = pager.querySelector('[data-bandit-status]');
    const prevButtons = Array.from(pager.querySelectorAll('[data-bandit-prev]'));
    const nextButtons = Array.from(pager.querySelectorAll('[data-bandit-next]'));
    const initialLevel = Number(new URLSearchParams(window.location.search).get('level'));
    let currentIndex = Number.isInteger(initialLevel) ? initialLevel : 0;

    const clampIndex = (index) => Math.min(Math.max(index, 0), pages.length - 1);

    const showPage = (index, shouldScroll = false) => {
      currentIndex = clampIndex(index);

      pages.forEach((page, pageIndex) => {
        page.hidden = pageIndex !== currentIndex;
      });

      tabs.forEach((tab) => {
        tab.classList.toggle('is-active', Number(tab.dataset.banditTab) === currentIndex);
        tab.setAttribute('aria-current', Number(tab.dataset.banditTab) === currentIndex ? 'page' : 'false');
      });

      prevButtons.forEach((button) => {
        button.disabled = currentIndex === 0;
      });

      nextButtons.forEach((button) => {
        button.disabled = currentIndex === pages.length - 1;
      });

      if (status) {
        status.textContent = `Bandit ${currentIndex} / ${pages.length - 1}`;
      }

      const url = new URL(window.location.href);
      url.searchParams.set('level', currentIndex);
      window.history.replaceState(null, '', url);

      if (shouldScroll) {
        pager.scrollIntoView({ behavior: 'smooth', block: 'start' });
      }
    };

    tabs.forEach((tab) => {
      tab.addEventListener('click', () => showPage(Number(tab.dataset.banditTab), true));
    });

    prevButtons.forEach((button) => {
      button.addEventListener('click', () => showPage(currentIndex - 1, true));
    });

    nextButtons.forEach((button) => {
      button.addEventListener('click', () => showPage(currentIndex + 1, true));
    });

    showPage(currentIndex);
  })();
</script>
